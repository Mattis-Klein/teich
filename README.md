# Teich v1.03.1

Desktop GUI for structured Gemara word editing and navigation.

Consumes wordmaps generated by Teich Word Mapper.

No OCR logic exists in this repository.

---

## Architecture

Teich (GUI)  
⬇ consumes JSON wordmaps  
Teich Word Mapper (OCR engine)

Repositories are fully separated.

---

## Stack

- PySide6
- pandas
- odfpy
- python-dateutil

---

## Run

python run_app.py

---

## Design Principles

- No OCR inside GUI
- Word-based data model
- True Gemara layout preservation
- Page-based workflow
- Minimal dependency bleed
